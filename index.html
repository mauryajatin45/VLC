About VLC Media Player Web Clone

VLC Media Player Web Clone is a lightweight, modern web-based video player inspired by the popular VLC Media Player. This project allows users to upload and play video files directly in their browser with essential playback controls, such as play/pause, volume adjustment, playback speed, seek controls, and fullscreen mode.

The application is designed to provide a seamless, intuitive video playback experience using JavaScript, HTML5, and CSS3. It also includes customizable keyboard shortcuts, allowing users to easily control playback through their preferred input method.

Key Features:

  • Upload and play video files directly from your device.
  • Basic playback controls (Play/Pause, Volume, Speed, Seek).
  • Fullscreen mode for an immersive experience.
  • Keyboard and mouse shortcuts for easy navigation.
  • Drag-and-drop support for video file uploads.

Technologies Used: HTML5, CSS3, JavaScript, Font Awesome
